## ü§ù Pod Collaboration

### Individual Work
- Each person **forks this repo** to their own GitHub account
- Complete all milestones on your own fork
- Check off tasks as you go using the checkboxes in each milestone file

### Pod Work
- One pod member creates a **shared pod repo** (instructions in Milestone 1)
- Each member is added as a collaborator
- Use the shared repo to practice pull requests, issues, and code review
- Track the pod's progress using the [Pod Checklist](pod-checklist.md)

### Pod Reviews
After each milestone, review at least one pod member's work:
1. Visit their fork, profile, or project repo
2. Open an **Issue** with constructive feedback
3. Use the title format: `EdMotion Pod Review: Milestone X - [Your Name]`

For data projects specifically, review:
- Is the analysis question clear?
- Are the findings easy to follow?
- Could someone reproduce the analysis from the README alone?
